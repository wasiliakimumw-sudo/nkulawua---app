Instructions
Required Column:
  • name - Beneficiary name (required)
Optional Columns:
  • beneficiary_type - "private" or "communal" (default: private)
  • phone - Phone number
  • village - Village name
  • scheme - Scheme name (Mangale, Nkala, Dodza, Milala)
  • country - Country name
  • tax_id - Tax ID
  • household_count - Number of households
  • credit_limit - Credit limit amount
  • payment_terms - Payment terms in days (default: 30)
  • is_active - True or False
Note:

If a beneficiary with the same name already exists, it will be updated with the new values.
